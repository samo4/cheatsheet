# Introduction

Control system is mechanism that alters the future state of a system. control theory is a strategy to select the approriate inputs.

Control theory is converting unstable open loop systems to stable closed loop systems, which are really just open loop systems again :).

You could just try out all the values, but that's not a particulary great method.
